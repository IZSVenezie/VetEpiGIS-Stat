VetEpiGIS-Stat

a QGIS plugin to make easy spatial statistics

View project on GitHub

VetEpiGIS-Stat is a member of VetEpiGIS QGIS plugin family. The goal of the development to collect spatial statistical methods are used in veterinary epidemiology into a plugin and create an easy to use interface for their application.
